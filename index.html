natUIve is your fast, light and functional framework

Plus it haz a picture of a cat

Increase conversion by drastically increasing page load speed to 1 second on slow mobile networks. Improve development speed by using standard building blocks that are flexible to customise and combine. Improve stability and compatibility by not messing with browsers any more than necessary. Go natUIve.

✓ Please do

  • Use a grid with arbitrary cell elements inside .row. Cell classes like .column23 respected, but not required.
  • Use vertical-alignment in cells, it's finally easy and dependable.
  • Embed any kind of element into any element suitable for container: Cell, List, Slider, Accordion.

✖︎ Please do not

  • Deviate from the base containers and <form> structure.
  • Use absolute positioning if possible.
  • Add bloatware libraries.
